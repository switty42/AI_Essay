# AI Essay
Docs 11-19-25 V1 (For questions or comments:  Stephen Witty switty@level500.com)  

### Project Overview:
The AI Essay project is an exploration into using modern AI models to create and classify essays.  The project includes tools for generating essay content and classifying essay topics.  The models used in the project include Claude, Gemini, GPT, and Grok.  This repository includes the source for the project and various test runs used for analysis.

This project was created in collaboration with Hannah Witty, whose insights helped shape the direction and design of the system.

### Usage:

- The software is a Python3 script and runs in the usual manner
- The script has been tested on Ubuntu
- Place your own API key toward the top of the Python script
- Edit the program constants as needed at the top of the Python script
